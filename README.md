## How to use

- Type in your Query

## Query Description
- FROM --> us to access the csv file
- SELECT --> select the column
- WHERE --> the conditions. (Ex: age <= 18)
- ORDER --> Order the printed list
- LIMIT --> Limit the printed list
- AND
- OR
- ASC --> Ascending order
- DESC --> Descending order
- BY
